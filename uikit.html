UI Kit & Style Guide

This page catalogs reusable UI patterns for the Micronesian Teachers Digital Library.

Use this reference to assemble new screens quickly: each pattern links to its original source and keeps the classes that our CSS and JavaScript expect.

Overview

The UI kit surfaces canonical markup from book.html, library.html, and login.html. Copy these blocks to stay aligned with the production design system and Laravel components.

Keep Proxima Nova as the primary typeface, reuse the spacing tokens defined in :root, and prefer existing utilities such as books-grid, section-list, and button-line before inventing variants.

Colors

Palette variables live in :root and fuel buttons, banners, and alerts. Pair them with existing utility classes instead of inline styles in production.
