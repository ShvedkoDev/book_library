### Form Field Priority

**Tier 1: Critical (Red Banner)**
1. Title * - Required, auto-focused
   - Validation: Required, max 500 characters, unique preferred

**Tier 2: Review Recommended (Yellow Banner)**
2. Subtitle - Copied, often needs edit
3. Publication Year - Copied, usually changes
4. Pages - Copied, may vary
5. Description - Copied, MUST review for accuracy

**Tier 3: Optional (Gray Banner)**
6. PDF File - Upload now or later
7. Thumbnail - Upload now or later

**Tier 4: Auto-Copied (Green Collapsed Section)**
- All relationships and classifications
- Settings and metadata
- Contact information

### Form Behavior

**Auto-save Draft**:
- Every 30 seconds
- After each field blur
- Indicator: "Draft saved 2 seconds ago"

**Keyboard Shortcuts**:
- `Tab` - Move to next field
- `Shift+Tab` - Move to previous field
- `Ctrl+S` / `Cmd+S` - Save draft
- `Ctrl+Enter` / `Cmd+Enter` - Save and continue
- `Esc` - Cancel (with confirmation)

**Validation**:
- Real-time validation on blur
- Red border + error message for invalid fields
- Green checkmark for valid required fields
- Block "Save & Continue" until required fields valid

**Goal**: Enable duplication without mouse

**Implementation**:
```
1. Navigate to book (Arrow keys or search)

5. Type new title
6. Press Tab â†’ Move to year
7. Type year
8. Press Ctrl+Enter â†’ Save and close
```

**Benefits**:
- Ultra-fast for experienced admins
- Reduces repetitive strain
- Enables batch processing

### Pattern 3: Smart Defaults

**Goal**: Reduce decisions needed

**Auto-selection based on context**:
- If book is in collection â†’ Copy collection relationships automatically
- If book has series number â†’ Auto-increment in title
- If book language is rare â†’ Copy all language metadata

- Learns from user behavior
