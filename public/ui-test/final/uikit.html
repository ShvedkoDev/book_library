Overview

This UI Kit contains all reusable components from the Micronesian Teachers Digital Library project. The design system uses WordPress CSS variables and styling standards to maintain consistency across all templates.

CSS Variables

  • --wp--preset--color--coe-green: #009877 - Primary brand color
  • --wp--preset--color--coe-blue: #005a70 - Secondary brand color
  • --wp--preset--color--coe-black: #0f1820 - Dark text
  • --wp--preset--color--coe-gray: #b1b6b9 - Muted text/borders
  • --wp--preset--font-family--proxima-nova - Primary font family
  • --wp--preset--spacing--* - Spacing scale (20-80)
