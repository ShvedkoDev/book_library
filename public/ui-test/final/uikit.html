Overview

This UI Kit showcases the design components used throughout the Micronesian Teachers Digital Library. The design system is built on WordPress CSS variables and follows academic/educational design principles.

How to Use

  • Reuse existing CSS classes from assets/css/wordpress-main.css
  • Follow WordPress CSS variable conventions (--wp--preset--*)
  • Maintain accessibility standards with proper labels and ARIA attributes
  • Use semantic HTML elements where appropriate
